About WhistleDrop

WhistleDrop allows whistleblowers to securely upload sensitive information while maintaining anonymity. The uploaded files are encrypted and can only be accessed by authorized journalists.

How It Works

  1. Whistleblower uploads a file through the secure platform.
  2. The file is encrypted and stored securely.
  3. Journalists can retrieve and decrypt the files using their private keys.
